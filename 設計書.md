# AtCoder Title Copy (ACTC) 基本設計書 (TypeScript & Rollup版)

## 1. 開発構成
* **言語**: TypeScript
* **ビルドツール**: Rollup
* **出力形式**: IIFE (Immediately Invoked Function Expression)
* **メタデータ管理**: `manifest.json` からビルド時にヘッダーを自動生成。

## 2. ディレクトリ構成（主要部分）
```text
src/
└─ actc/
   ├─ main.ts          # エントリポイント（DOM監視・初期化）
   ├─ button.ts        # ボタン要素の生成・スタイル適用
   ├─ extractor.ts     # ページ情報（タイトル・コンテスト名）の抽出ロジック
   ├─ clipboard.ts     # クリップボード操作・フィードバック処理
   └─ manifest.json
```

## 3. モジュール別詳細設計

### 3.1 `extractor.ts` (情報抽出層)
AtCoderのDOM構造から情報を取得し、型安全に処理します。
* **getContestId()**: `window.location.pathname` から正規表現でコンテストIDを取得し、`.toUpperCase()` で整形。
* **getTaskInfo()**: `.h2` クラスを持つ `<span>` 要素の `innerText` を取得。「記号 - タイトル」の形式を分割して保持。
* **formatTitle()**: 上記を結合し `ABC123A - Five Antennas` 形式の文字列を生成。

### 3.2 `button.ts` (UI生成層)
* **createCopyButton()**:
    * `HTMLButtonElement` を作成。
    * `className` に `btn btn-default btn-sm` を付与。
    * `style.marginLeft = "10px"` を適用。
    * 初期テキストとして `Copy Title` を設定。

### 3.3 `clipboard.ts` (アクション層)
* **copyToClipboard(text: string, button: HTMLButtonElement)**:
    * `navigator.clipboard.writeText(text)` を実行。
    * 成功時、`button.innerText` を `Copied!` に変更。
    * `setTimeout` (2000ms) で元のテキストに復帰。

### 3.4 `main.ts` (制御層)
* **DOMContentLoaded** またはスクリプト実行時に実行。
* `document.querySelector('.h2')` でターゲット要素を特定。
* ボタンをターゲット要素の末尾（`appendChild`）に挿入。

すでにしてあるため省略。

## 5. 技術仕様・型定義
| 項目 | 指定内容 |
| :--- | :--- |
| **tsconfig** | `target: "ESNext"`, `module: "ESNext"`, `strict: true` |
| **DOMアクセス** | `document.querySelector` 等の戻り値に対する非nullチェックの徹底 |
| **外部ライブラリ** | 基本なし（標準Web APIのみ使用） |

## 6. HTML挿入イメージ

設計に基づき、以下のような構造を動的に生成します。
```html
<span class="h2">
  A - Five Antennas
  <button class="btn btn-default btn-sm" style="margin-left: 10px;">Copy Title</button>
</span>
```
